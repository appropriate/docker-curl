# docker-curl
Alpine-based image with just curl

## Usage

```console
$ docker run --rm appropriate/curl -fsSL https://www.google.com/
```

## Tags

* `appropriate/curl:latest`: based on `alpine:latest`
* `appropriate/curl:edge`: based on `alpine:edge`
